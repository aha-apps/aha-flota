# AHA Flota 🚛

Control de vehículos y flotilla offline-first.

## Módulos

- **Vehículos** — CRUD con foto, búsqueda por placas/número económico
- **Combustible** — Registro de cargas, cálculo km/litro, gráfico Chart.js
- **Mantenimiento** — Servicios, próximo mantenimiento por km, alertas
- **Incidentes** — Multas, accidentes, averías
- **Reportes** — Dashboard con KPIs, gráficos y export CSV

## Perfil

Lite — funciona con doble clic en index.html (file://)

## Stack

- Alpine.js + Dexie.js + CryptoJS + Chart.js
- DaisyUI 4 + Tailwind CSS + Bootstrap Icons + Animate.css
